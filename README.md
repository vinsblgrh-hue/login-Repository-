# Login System Project (Python + MySQL)

## Description
This is a simple login system project developed using Python and MySQL. It allows users to register and login by storing and verifying credentials from a database.

## Features
- User registration
- User login authentication
- Password verification using MySQL database
- Simple console-based interface

## Technologies Used
- Python
- MySQL

## Database
- MySQL database is used to store user credentials
- Table includes fields like username and password

## How to Run
1. Install Python
2. Install MySQL and create database
3. Run the Python file
4. Enter username and password in console

## Future Improvements
- Add password encryption (hashing)
- GUI interface
- Forgot password feature
